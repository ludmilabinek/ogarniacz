Anyone with this URL can read your accepted events. Share with your spouse or co-parent only.

How to subscribe

Google Calendar: Settings → Add calendar → From URL — paste the URL above.

Apple Calendar: File → New Calendar Subscription — paste the URL above.

Outlook: Add calendar → Subscribe from web — paste the URL above.

Thunderbird: File → New → Calendar → On the Network — paste the URL above.

Google Calendar limitations. Google polls subscribed calendars every 12–24 hours (sometimes up to 48), with no way to speed this up from the client side. Google also does not show reminders embedded in subscribed calendars — to receive the morning-of-day-before reminder, set a per-calendar default reminder in Google Calendar's own settings. Apple Calendar, Outlook, and Thunderbird honor both the polling cadence hints and the embedded reminder fully.
